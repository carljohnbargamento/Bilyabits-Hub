Bilyabits Hub

About

Bilyabits Hub is a Facebook chat bot framework powered by ws3-fca, designed for easy automation and management of chat commands on Messenger. It is modular, extensible, and comes with a set of built-in and custom command handling features. The bot notifies admins on successful logins, supports automatic bio updates, and is simple to set up for your own usage.

What It Does

  • Automates Facebook Messenger chat interactions and responses.
  • Supports custom commands, event listening, and modular expansions (via JavaScript).
  • Automatically updates bot bio and notifies admins when online.
  • Features robust error handling and regular session refreshing.
  • Easy configuration and quick setup for personal or community bots.
